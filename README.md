### 一键安装

#### 新建文件夹
~~~
mkdir YunzaiBot && cd YunzaiBot
~~~
#### 安装
~~~
wget https://raw.githubusercontent.com/titeeapps/YunzaiBot_OneKeyInstall/main/run.sh && chmod +x run.sh && bash run.sh
~~~
安装过程中可随时按 `ctrl+c` 来取消脚本,但安装过程不会回滚,建议取消安装后删除安装文件夹.
